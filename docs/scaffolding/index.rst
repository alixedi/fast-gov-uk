Scaffolding for common service patterns
=======================================

In addition to the implementation of GOV.UK Design System in Python, fast-gov-uk includes
many common service patterns like error pages, cookie banner, phase banner, a footer and even
a feedback form.

You have seen the example for creating a simple page. In addition, fast-gov-uk also provides
a minimal API to quickly build forms and question pages.
